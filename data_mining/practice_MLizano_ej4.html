Tenemos una inmensa cantidad de variables, muchas se ven claramente correlacionadas, sobretodo (y como era de esperarse) las de permetros, reas y radios, pero tambin hay muchas otras con correlaciones fuertes, pero es importante empezar a reducir la cantidad de variables para que sea un poco ms fcil el anlisis tanto de las correlaciones, como del modelo que queremos generar. Tambin vemos que se trata de variables "repetidas" pero con mediciones diferentes: para cada valor tenemos la media, el error estandar y el worst case, todo como variables separadas, por lo que tambin es normal que estn altamente correlacionadas entre s estas variables.

3. Sabiendo que se trata de predecir la variable binaria diagnosis (‘M,’B’). ¿Cual sería el modelo adecuado de predicción? ¿Qué problemas auguras a la luz de la información sobre las correlaciones del archivo?¶

Considerando que la variable objetivo es binaria, el modelo adecuado para predecirla sera una regresin logstica; sin embargo como se mencion anteriormente, hay muchsimas variables, por lo que impide un buen anlisis, adems de que muchas de ellas tienen una alta correlacin entre s; por lo que aplica en este caso es una reduccin de dimensionalidad antes de empezar con la creacin del modelo final, con esto vamos a eliminar el ruido de variables casi repetidas o con poca influencia sobre el modelo y podemos tener un set de datos mucho ms resumido e interpretable.

4. Como posible estrategia se plantea la realización de un ACP para la reducción de dimensiones y un modelo de predicción adecuado utilizando como predictores las componentes principales resultantes de tal forma que se retenga al menos el 70% de la variabilidad del archivo.¶

a. Valora la adecuación muestral a priori¶

Ya habamos realizado la matriz de correlacin, que es uno de los pasos de la adecuacin muestral para determinar que los datos son adecuados para una reduccin de dimensiones, ya vimos en dicha matriz que efectivamente parece haber muchas correlaciones, por lo que procedemos con:

  • Test de esferidad de Bartlett: (hiptesis nula es que la matriz de correlaciones es igual a la matriz identidad, por lo que no hay correlaciones significativas entre las variables)
  • ndice de KMO: para contrastar la correlacin entre pares de variables y sus correlaciones parciales, buscamos que sea > 0.7
